rviz is a 3D visualizer for the Robot Operating System (ROS) framework.

For more information, please see the wiki: http://wiki.ros.org/rviz

Maintainers:
- Robert Haschke (2019-)
- William Woodall (2013-2018)
- David Gossow (2013)
- Dave Hershberger (2011-2013)
- Josh Faust (2010)
